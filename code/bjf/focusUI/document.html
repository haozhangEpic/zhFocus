bjf focusUI模块
该模块将提供管理光标框的显示和隐藏，移动，调整大小等一系列功能和对象。请参考demo演示。
光标框系统定义了一系列光标框事件，具体参见focusUI事件列表
方法和对象列表：
  • bjf.focusUI
  • bjf.focusUI.init
  • bjf.focusUI.on
  • bjf.focusUI.lost
  • bjf.focusUI.setSize
  • bjf.focusUI.setOffset
  • bjf.focusUI.setPostBy
  • bjf.focusUI.setGap
  • bjf.focusUI.setPost
方法和对象说明：
bjf.focusUI
bjf.focusUI是一个控制光标框的显示和隐藏，移动，调整大小等一系列功能的对象
bjf.focusUI.init(border_width_, class_name_, user_style_)
初始化光标框，设置外框大小
@param border_width_  [可选，默认为4]光标边框宽度[int类型]
@param class_name_ [可选]光标框的样式名称，通过样式可定义如阴影，圆角，颜色等[string类型]
@param user_style_ [可选]光标框DIV的的style属性对象，可直接指定阴影，圆角，颜色等样式[object类型]
                
bjf.focusUI.init();
//bjf.focusUI.init(4, 'focus');  //指定光标框的宽度为4，className=focus
//bjf.focusUI.init(6, '', {
    borderColor:'#ff0000', 
    borderStyle: solid;
    borderRadius:'20px'
});//指定光标框的宽度为6，光标框颜色为红色，20px的圆角
bjf.focusUI.on()
将光标框设置为可见
                
bjf.focusUI.on();
bjf.focusUI.lost()
隐藏光标框
                
bjf.focusUI.lost();
bjf.focusUI.setSize(w, h)
设置光标框的大小
@param w 光标框的宽度[string类型]
@param h 光标框的长度[string类型]
                
bjf.focusUI.setSize(7, 9);
bjf.focusUI.setPostBy(x_, y_)
计算光标框的位置
@param x_ 光标节点x坐标[int类型]
@param y_ 光标节点y坐标[int类型]
                
bjf.focusUI.setPostBy(0, 0);
bjf.focusUI.setGap(x_, y_)
设置光标集合下光标框之间的间距
@param x_ 光标框之间x轴上的间距[int类型]
@param y_ 光标框之间y轴上的间距[int类型]
                
bjf.focusUI.setGap(0, 0);
bjf.focusUI.setPost(x, y)
设置光标框的位置
@param x_ 光标框x轴上的位置[int类型]
@param y_ 光标框y轴上的位置[int类型]
                
bjf.focusUI.setPost(0, 0);
